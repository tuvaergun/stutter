# stutter
Seed-based wordlist generator

# Description
This script takes a single string as an argument and outputs all possible permutations based on a text file of similar characters.

===

# Example
Input:

`./stutter.py asdf`



Output:

```
asdf
asdF
asDf
asDF
aSdf
aSdF
```

...

```
@5Df
@5DF
@$df
@$dF
@$Df
@$DF
Seed 'asdf' generated 48 words
```
